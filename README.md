# Quote of the Day

## Description
A CLI applicaiton that fetches random inspirational quotes from the ZenQuotes API and allows users to save their favorite quotes.

## Features
- Fetch random quotes from the ZenQuotes API
- Save a quote and the author if user chooses to
- View saved quotes with authors

## How to run:
install in bash: pip install requests

# Code Structure
- 'requests' library to call API
- List of dictionaries to store favorite quotes with their author
- JSON pasing with .json()
- Error handling for API failures
